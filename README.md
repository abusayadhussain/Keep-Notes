# Keep-Notes
This app will allow user to keep notes and it will save to a listview. when user click the note they can also edit the notes and When they wants to add a new note they can simply tap on the manu bar and select add notes and start typing the note and after writing when they back from there it will be saved on the listview. If they wants they can long press the listed item and delete the note as well.
